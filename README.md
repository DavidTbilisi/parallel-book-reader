# Parallel book reader

web app, intended to learn languages through reading. 

## Options
- [ ] read from api
- [ ] read from txt files
- [ ] read from HTML files
- [ ] show 2-3 books at the same time
- [ ] syncronized scrolling
- [ ] ?add unknown words to dictionary
...
